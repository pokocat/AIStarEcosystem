## 🎨 用户交互流程

### 新的完整流程

```
1. 进入AI歌手孵化器
   ↓
2. 看到主页：
   - 4个统计卡片（总数/活跃/作品/粉丝）
   - 搜索框 + 筛选按钮
   - 已孵化的歌手卡片网格
   ↓
3. 两种操作：
   
   A. 编辑现有歌手：
      点击歌手卡片的"编辑"按钮
      ↓
      进入SingerEditor编辑器
      ↓
      选择创作模块：
      - 官方IP库（快速应用预设）
      - 参数调节（精细调整）
      - 基因混合（混合创造）
      - 图片定制（上传参考）
      - 服装换装（200+服装）
      - 姿态动作（30+姿态）
      ↓
      点击"保存"返回主页
   
   B. 创建新歌手：
      点击"创建新歌手"按钮
      ↓
      自动创建草稿并进入编辑器
      ↓
      （同上编辑流程）
      ↓
      保存后出现在主页
```

## 📊 数据结构

### Singer接口
```typescript
interface Singer {
  id: string;                    // 唯一标识
  name: string;                  // 名称
  avatar: string;                // 头像URL
  style: string;                 // 音乐风格

  quality: 'common' | 'rare' | 'epic' | 'legendary';  // 品质
  createdAt: Date;               // 创建时间
  stats: {                       // 统计数据
    songs: number;               // 作品数
    fans: number;                // 粉丝数
    popularity: number;          // 人气值
  };
  tags: string[];                // 标签列表
}
```
