## 🐛 常见问题排查

### 问题1: 组件不显示
```
检查：
1. AIIncubator组件是否正确导入
2. activePage状态是否为'persona'
3. 所有必需的props是否传递
4. 浏览器控制台是否有错误
```

### 问题2: 样式错乱
```
检查：
1. Tailwind CSS是否正确加载
2. 自定义CSS是否覆盖
3. 浏览器缓存是否清除
```

### 问题3: 图片不加载
```
检查：
1. Unsplash图片URL是否可访问
2. 网络连接是否正常
3. 浏览器是否阻止外部图片
```

### 问题4: 参数不同步
```
检查：
1. personaParams状态是否正确传递
2. setPersonaParams函数是否正常工作
3. 控制台是否有状态更新日志
```
