## 🐛 已知问题 & 注意事项

### 1. Supabase 连接问题
- **状态**: 403 权限错误
- **影响**: 所有数据无法持久化
- **临时方案**: 使用本地状态管理（useState）
- **解决方案**: 需要重新配置 Supabase 项目权限

### 2. 艺人数据同步
- **当前**: 签约后艺人仅存储在前端状态
- **问题**: 刷新页面后数据丢失
- **待实现**: Supabase 实时同步

### 3. 图片资源
- **当前**: 使用 Unsplash CDN
- **建议**: 迁移到自有 CDN 或 Supabase Storage
- **原因**: 稳定性和加载速度
