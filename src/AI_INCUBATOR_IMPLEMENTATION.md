## 💡 用户使用流程

### 新手路径（推荐）
```
1. 进入孵化器 → 默认显示"官方IP库"
2. 浏览10个精品AI歌手
3. 点击"立即使用"
4. 跳转到工作室开始创作音乐
```

### 中级路径
```
1. 切换到"参数调节"标签
2. 调整3个基础滑块（视觉/性格/音乐）
3. 查看右侧实时预览
4. 点击"生成预览"
5. 满意后保存到"我的艺人"
```

### 高级路径（最有趣）
```
1. 切换到"基因混合"标签
2. 选择父本A（如：霓虹战士）
3. 选择母本B（如：云裳仙子）
4. 调整基因配比滑块（50/50）
5. 开启5%随机突变
6. 点击"开始实验"
7. 等待3秒生成结果
8. 查看稀有度（可能获得传说级角色！）
9. 保存到艺人库
```
